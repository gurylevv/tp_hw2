# Система управления рецептами

Проект для управления рецептами. Домашнее задание №2 по предмету "Технологии программирования".

## Установка

1. Склонируйте репозиторий на свой компьютер:
   `git clone https://github.com/gurylevv/tp_hw2`
2. Перейдите в папку проекта:
   `cd tp_hw2`
3. Установите зависимости:
   `pip install -r requirements.txt`
4. Запустите автоматические тесты:
   `pytest`
